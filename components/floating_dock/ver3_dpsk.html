Floating Navigation Dock

A sleek, responsive floating dock with smooth animations and dark/light mode. Perfect for modern web applications.

About This Component

This floating dock provides a clean navigation solution that stays accessible at the bottom of your screen.

Hover over the items to see them expand with smooth animations and display helpful tooltips.

Toggle between dark and light mode using the button in the top right corner.

Features

• Responsive design that works on all devices

• Smooth animations and transitions

• Dark/light mode with persistent setting

• Clean, modern design with glassmorphism effects

• Easy to customize and extend

Created with pure HTML, CSS, and JavaScript • No frameworks required
